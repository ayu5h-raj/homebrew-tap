# Homebrew Tap

This is a [Homebrew](https://brew.sh/) tap for my personal projects.

## Installation

```bash
brew tap ayu5h-raj/tap
```

## Available Casks

### QuickCSV

High-performance CSV viewer for macOS. Handles files from 100MB to 2GB+ with zero lag.

```bash
brew install --cask quickcsv
```

**Features:**
- Memory-mapped file loading (zero-copy)
- Progressive loading with instant display
- Fast search across all columns
- JSON viewer for embedded JSON fields
- Dark/Light mode
- Column sorting

[GitHub Repository](https://github.com/ayu5h-raj/quickcsv)
